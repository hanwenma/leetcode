# 目录

## code
用于存放实现的算法代码

## test
用于存放针对 code 目录中对应的 jest 测试用例

# 配置 babel
- **https://www.babeljs.cn/docs/usage**

# 配置 jest
- **https://www.jestjs.cn/**
- 通过 `npm install core-js` 解决报错 `Cannot find module 'core-js/modules/es.regexp.exec.js' from xxx`

# 执行测试用例
`npm run test`

# 提交代码
- `git add .`
- `npm run  commit`
- `git push`
